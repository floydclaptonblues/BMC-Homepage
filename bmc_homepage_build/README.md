# Balcony Music Club Homepage Prototype

Open `index.html` in a browser to review the homepage concept.

Included:
- Cropped/tiled BMC hero GIF background that removes the old bottom taskbar area.
- Live Tonight hero card.
- Upcoming Shows, BMC Radio, Host Your Event, Gallery, Signup, and Find Us sections.
- JazzyCat quartet sprite treatment with CSS marching/flip animation.
- Uploaded venue photos used as the small gallery cells.
- Responsive layout with mobile navigation.

Files are standalone HTML/CSS/JS and do not require a build step.
